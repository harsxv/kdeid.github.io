Setelah install Mate Desktop di notebook lenovo dan elementary, ternyata setelah masuk kedalam environment, tombol NumLock selalu dalam keadaan ON, jadi beberapa huruf di keyboard diterjemahkan sebagai angka saat input.

Terus gimana? Googling-googling nemu solusi, salah satunya dari link ini. Cuma masalahnya, disitu buat set dalam keadaan ON saat startup, yang dicari itu harus dalam keadaan OFF. Kalau mau default ON saat startup caranya seperti ini:

$ sudo apt-get install numlockx
$ gksudo gedit /etc/lightdm/lightdm.conf

Terus tambah baris ini di lightdm.conf

greeter-setup-script=/usr/bin/numlockx on

Tapi karena disini pengen NumLock nya OFF saat startup, ganti on jadi off terus hasilnya jadi begini:

greeter-setup-script=/usr/bin/numlockx on

Cara ini juga bisa dicoba untuk Desktop Environment lain.
