Ceritanya gue baru jatuh cinta dengan terminal, bukan, bukan gue jatuh cinta di terminal atau kangen pacaran sama terminal dengan petugas-petugasnya. Tapi karena nyaman dengan linux setelah install beberapa aplikasi yang berjalan di terminal.

Apa sih terminal? Pernah pake DOS, atau command prompt? Nah kalau di linux, itu disebut terminal. Ada juga sih distro lain yang menyebutnya console, bahkan konsole kalau desktopnya KDE.

Terminal

Kalau kata orang, gue terlihat lebih 'geek' banget setelah pake aplikasi di terminal, terlihat lebih keren. Padahal ada desktop environment yang sebenernya jalanin perintah terminal. Jadi ini punya Destkop buat jalanin Terminal buat buka Terminal. Jadi inilah list aplikasi yang sudah di-install.

MOC - Music On Console

[caption id="attachment_2306" align="alignnone" width="625"] Music on Console[/caption]

Download diÂ http://moc.daper.net/download terus baca readme buat install. Buat jalaninnya, buka terminal lalu ketik

mocp

terus enter.

Lynx - Browsing Mode Text

[caption id="attachment_2308" align="alignnone" width="625"] Lagi buka twitter pake lynx di terminal.[/caption]

Jadi lynx ini semacam web browser, emang web browser, sayangnya gak support javascript dan kawan-kawannya. Cara installnya gimana? Kalau di ubuntu, cukup pake perintah

sudo apt-get install lynx

Buat info lainnya bisa dilihat di http://lynx.isc.org/lynx2.8.7/index.html

Mutt - Email Client

[caption id="attachment_2310" align="alignnone" width="625"] Gmail di mutt, baca email di mutt, pengen permen di mutt.[/caption]

Ceritanya udah bosen sama gmail web atau bahkan mozilla thunderbird, ternyata ada juga email client text base, akhirnya ya install mutt, terus dipake buat kirim terima email. Cara install?

apt-get install mutt

Kalau kurang puas, bisa baca-baca detailnya disini http://mutt.blackfish.org.uk/
