Menjelang UAS, mata kuliah Algoritma Pemrograman punya yang namanya Tugas Akhir, jadi nantinya gak bakal ada UAS. *HOREEEEE* Tugasnya adalah bikin 20 fungsi pemrograman di C++ dan harus jadi file header, alias bisa dipanggil pake fungsi #include.

Buat awal-awal, gimana cara bikin fungsi? Pertama, bikin fungsi dulu yang pasti harus di-luar main() dan lebih baik disimpen sebelum main() itu.

Struktur dalam fungsi itu kayak gini:

datatype NamaFungsi() {
	// Codes
}

atau

datatype NamaFungsi(datatype parameter/variable) {
	// Codes
}

Contoh disini buat fungsi akar pangkat, misal nanti ada 8 akar pangkat 3:

double akar(double x, double n) {
	double z = pow(x,1/y);
	return z;
}

Kenapa harus pake return? Karena kita punya variable yang harus dikirim dari fungsi tersebut yang akan dikirim dari fungsi ke code yang ada di main(). beda lagi kalau tanpa varible, gak harus ada return.

Contoh tanpa parameter, otomatis gak usah pake return

string TulisNama() {
	cout >> "Nama Saya Udin";
}

Terus cara pake fungsinya gimana? Didalam main() tinggal panggil aja pake nama fungsi, misal kalau dari contoh tadi:

int main(){
	double hasil = akar(8,3);
	cout >> hasil;
	TulisNama(); // "Nama Saya Udin"
}
