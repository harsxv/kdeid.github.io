Demam desain simple kotak-kotak sedang melanda dunia..!!! Sampai microsoft keluarin MetroUI buat windows 8 dengan desain kotak-kotak pula. Eh ternyata desain ini banyak disukai user di seluruh dunia, hingga Sergey Vimenov yang berkebangsaan Ukraine beserta timnya membuat framework terbaru, namanya Metro UI CSS.

Yang unik dari framework ini adalah tagline mereka yang menyebutkan

This is not a theme or clone for Twitter Bootstrap !!!

Ya memang sih, seklias mirip Twitter Bootstrap, dari mulai struktur sampai dokumentasi. Apakah itu salah satu tujuan agar user mudah beralih dari twitter bootstrap ke Metro UI CSS ini? Atau Metro UI CSS sebenarnya merupakan turunan dari Twitter Bootstrap? Hanya pengembang dan tuhan yang tau akan hal ini. :D

Jadi menurut saya, Metro UI CSS ini merupakan framework yang mudah digunakan, dan yang sudah mengenal twitter bootstrap tidak akan merasa canggung, karena struktur yang hampir sama, namun sayangnya dokumentasi Metro UI CSS ini tidak sebaik Twitter Bootstrap. Tapi secara keseluruhan, framework ini bisa dibilang "KEREN".

Mau coba? Langsung aja kesini http://metroui.org.ua/ atau lihat projectnya di sini https://github.com/olton/Metro-UI-CSS
