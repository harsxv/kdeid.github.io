Pengalaman instal ArchLinux di MacBook White Pak tuanpembual, sekarang baru sempat nulis buat berbagi. Mungkin pernah lihat Pak tuanpembual atau temen-temen blankOn pake kaos dengan tulisan mac.di.blankon.in dan merasa keren, pengen install Linux di Mac? PENGEN? Pasti jarang yang pengen.... :(

Unduh file ISO Linux

Unduh dulu file ISO Linux dari distro manapun, misal ubuntu atau openSUSE, arch juga boleh.

Bakar file ISO

Bisa bakar file ISO ke media CD / DVD jika ada disc writer, kalau mau pake bootable USB, bisa dengan perintah dd, untuk perintah dd, harap ikuti langkah dibawah
Cari dulu UFD di direktori disk yang mana, lalu unmount.

$ diskutil list

Misal ada di disk3, maka

$ sudo diskutil unmountDisk /dev/disk3

Setelah itu, baru bisa gunakan dd untuk menulis file ISO ke UFD, misal seperti ini:

$ sudo dd if=/direktoridownload/fileLinux.iso of=/dev/disk3

Unduh refind

Unduh dulu refind di http://www.rodsbooks.com/refind/ lalu extract di folder Home, setelah itu buka Terminal, buka direktori hasil extract tadi, dan jalankan perintah

OSX Sebelum El Capitan

Langkah di bawah ini khusus buat OSX sebelum El Capitan, berarti mulai dari Yosemite kebawah ya :D

$ sudo ./refind-install

Selesai? Lanjut restart Mac.

Untuk El Capitan

Karena ada restriksi di El Capitan, harus lewat 'Recovery Mode' dulu untuk install refind, caranya?

  1. Restart Mac, tekan dan tahan tombol Command + R di keyboard saat proses restart
  2. Setelah masuk ke Recovery Mode buka Terminal di "Utilities -> Terminal"
  3. Ketik csrutil disable lalu Enter
  4. Restart lagi Mac nya, masuk lagi ke Recovery Mode dengan cara yang sama, tekan tahan tombol Command + R
  5. Buka lagi Terminal di "Utilities -> Terminal"
  6. Buka folder hasil extract refind tadi dengan perintah # cd /Volumes/Macintosh\ HD/Users/namauser/
  7. Ketik perintah ./refind-install dan tunggu hingga selesai.
  8. Restart Mac.

Installasi Linux

Setelah restart, layar mac dengan tampilan refind akan terbuka, pilih boot dari USB, biasanya ditandai dengan gambar penguin atau logo USB, selanjutnya adalah konfigurasi partisi harddisk dan dilanjut dengan instalasi Linux seperti biasa.

CATATAN

Disarankan untuk mengatur partisi harddisk sebelum instalasi linux agar lebih memudahkan proses.

Karena instalasi Linux akan menghapus refind, dan masalah ini membuat kita tidak bisa masuk ke OSX karena tidak terdaftar di GRUB, maka perlu menginstal kembali refind dengan cara:

  1. Restart Mac nya, masuk ke Recovery Mode dengan cara tekan tahan tombol Command + R.
  2. Buka Terminal di "Utilities -> Terminal".
  3. Buka folder hasil extract refind tadi dengan perintah # cd /Volumes/Macintosh\ HD/Users/namauser/.
  4. Ketik perintah ./refind-install dan tunggu hingga selesai.
  5. Restart Mac.
  6. Sekarang muncul pilihan untuk boot Linux dan OSX.
