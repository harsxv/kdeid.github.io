Lagi ngerjain kerjaan yang sebenernya udah lama harus dikerjain sejak dulu tapi gak dikerjain juga. Di rumah temen, pake koneksi speedy, dan sepertinya udah gak inject script iklan lagi. Etapi, kenapa suka redirect ke linkbucks? Awalnya ngira ini browser add-on atau malware... Tapi, aku kan pake linux, dan selama ini install aplikasi dari pacman / AUR, coba di beda browser juga masih sama. Jadi kesimupalannya adalah: Speedy!

Speedy masih kurang ajar, entah apa yang ada dipikiran petingginya, konsumen bayar, masih harus diporotin lagi bandwidth yang gak begitu kenceng buat liatin iklan. Screw you!

Berhubung gak bisa ganti DNS di modem, yasudah ganti DNS di client side aja :D

Caranya? Cari network settings / wifi settings, terus Configure / edit. Kalau punyaku sih gini:

Kalau di gnome, tinggal klik tombol Gear di active network sekarang, nanti muncul kayak gini:

Masuk ke kolom IPv4, terus DNS Automatic nya matiin (OFF) *itu lupa di SS gak dimatiin. Terus Server nya edit, kalau ogut pake IP DNS punya google 8.8.8.8 sama 8.8.4.4.

Sudah? Terus Apply, terus lanjut browsing.

Selamat mencoba dan jangan sampai Speedy maksa kita buat nyariin recehan buat dia!
