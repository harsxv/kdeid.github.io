Biasanya kalau kita lagi edit dokumen dan butuh gambar dari dokumen lain kadang susah buat extract gambar yang ada di dokumen, bisa aja sih copy paste langsung via clipboard. Nah masalahnya kalau gak bisa? Bingung juga sih, masa iya capture screenshot terus crop. Ini bisa juga sih.

Caranya ternyata gampang, syaratnya dokumen harus format Open Document atau format punya Microsoft Office 2007 keatas (yang ada akhiran x di tiap ekstensinya). Misal kalau dokumen formatnya Microsoft Office 2003 kebawah, tinggal buka pake LibreOffice, save as ke Open Document Format dulu.

Caranya adalah:

  1. Copy dokumen jadi file baru, langsung aja Ctrl + C terus Ctrl + V.
  2. Rename ekstensinya jadi .zip, misal document.ods jadi document.ods.zip atau document.zip.
  3. Extract file zip nya, terus cari gambarnya.
  4. Untuk file Microsoft Office terbaru, ada di folder xl/media
  5. Untuk Open Document Format, ada di folder Pictures kalau file gambar, sisanya menyesuaikan lah, tinggal dicari kok.

Udah sih gitu doang, selamat mencoba dan jangan lupa baca do'a.
