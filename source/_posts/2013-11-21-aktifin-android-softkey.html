Ini gara-gara tadi pagi touch panel yang isinya menu sama back gak jalan di HP, emang ringkih sih ini panel, akhirnya punya ide buat "memanggil softkey dari kubur. Ada yang gak ngerti Android Softkey itu apa? Itu lho, tombol yang ada di layar, yang mirip HP nexus yang baru. Ini nih:

Nah, ngerti kan? Terus tujuannya biar layarnya jadi begini nih:

Nah, caranya sih gampang, asal device udah di-root, ada text editor sama file browser yang bisa akses ke folder /system/ seperti Root Browser atau Root Explorer.

Nah, sekarang buka folder /system/ terus buka file build.prop terus tambah baris kode ini di ujung aja:

qemu.hw.mainkeys=0

Terus save, restart, nikmati hasilnya :D

Nah, sekarang kalau mau disable atau mematikan tombol-tombol di device kaya tombol back, menu sama home biar gak bentrok, buka folder /system/usr/keylayout/ terus buka file Generic.kl, cari baris ini:

key 158   BACK
key 139   MENU
key 172     HOME

terus kasih tanda pagar di depannya, jadinya begini:

# key 158   BACK
# key 139   MENU
# key 172       HOME

Selamat mencoba dan jangan lupa restart device-nya :)
