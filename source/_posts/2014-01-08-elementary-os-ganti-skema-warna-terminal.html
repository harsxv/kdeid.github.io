Karena background hitam dan foreground putih di terminal itu gak jelas dilihat kalau pake proyektor, jadinya sebelum presentasi mikir dulu gimana caranya biar background putih, foreground hitam. Mulai dari install elementary tweak tools tapi gak puas, sampe akhirnya nemu cara edit pake dconf-tools.

Jadi terminal di elementary ini gak punya menu preference buat edit tampilannya kaya gnome-terminal atau yang lainnya.

Cara pertama, buka terminal terus install dconf-tools. Caranya? Piye masa gak tau.

$ sudo apt-get install dconf-tools

Terus panggil dconf-editor, masih di terminal, tinggal ketik

$ dconf-editor

lalu enter.

Buka org → pantheon → terminal → settings disitu tinggal edit value background, cursor-color, foreground.

Sudah? Terus buka aja terminalnya.
