Sering pake terminal di linux? Bukan terminal Kelas A Tasik atau terminal Giwangan lho. Contohnya ngetik apt-get install blablabla gitu. Kesel kan kalau ternyata satu perintah itu panjang banget? Sekarang kita bikin perintahnya jadi singkat dengan alias. Caranya?

Ini jelasin fungsi alias dulu ya, trick sebenarnya masih ada dibawah :D.
Buka terminal, terus ketik alias namaalias='perintah yang akan dipersingkat'. Contoh:

$ alias install='sudo apt-get install'
$ alias uninstall='sudo apt-get remove'
$ alias update='sudo apt-get update'
$ alias upgrade='sudo apt-get upgrade'

Jadi nanti kalau mau install, tinggal ketik install namapackage, misal:

$ install build-essential

Itu penjelasan penggunaan alias, selanjutnya edit file .bashrc via terminal

$ gedit .bashrc

terus masukan konfigurasi alias yang sudah atau mau ditambahkan di baris paling akhir, kalau perlu kasih komen diatasnya biar gak pusing nyarinya. Contoh:

# Custom Alias
alias install='sudo apt-get install'
alias uninstall='sudo apt-get remove'
alias update='sudo apt-get update'
alias upgrade='sudo apt-get upgrade'

Nah, sekarang manggil perintah atau contohnya mau install tinggal panggil aja aliasnya, gak ribet kan? :D
