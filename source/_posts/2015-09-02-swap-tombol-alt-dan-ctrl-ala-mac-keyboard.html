Niatnya cuma iseng, pengen coba ngetik ala pengguna macintosh, macbook, atau simplenya mac keyboard. Emang gimana sih? Jadi kalau di keyboard mac, tombol Alt dan Ctrl itu tukeran tempat, alias tombol Alt ada di tombol Ctrl dan sebaliknya. Jadi biar saya tampilkan gambarnya.

Ini keboard Mac English (US)

[caption id="attachment_4002" align="alignnone" width="700"] Keyboard Mac en_US[/caption]

Sebagai catatan, cara ini baru berlaku untuk keyboard standar, untuk laptop beda lagi karena ada tombol Fn yang letaknya beda, laptop saya tombol Fn diapit tombol Ctrl dan tombol Super, untuk keyboard Macbook sendiri tombol Fn berada paling luar.

Yang diperlukan adalah membuat file .Xmodmap dengan perintah $ gedit ~/.Xmodmap dan isikan konfigurasi ini:

clear control
clear mod1
clear mod4

! LEFT
keycode 37 = Alt_L Meta_L
keycode 64 = Control_L


! RIGHT
keycode 108 = Control_L
keycode 105 = Alt_L Meta_L

! ADD CONTROL
add control = Control_L Control_R
add mod1 = Alt_L Alt_R Meta_L Meta_R

Jika sudah selesai, reload konfigurasi xmodmap dengan perintah $ xmodmap ~/.Xmodmap
