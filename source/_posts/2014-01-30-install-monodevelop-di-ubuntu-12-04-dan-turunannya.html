MonoDevelop itu IDE OpenSource untuk bahasa pemrograman Mono dan .NET Framework, contohnya C# (dibaca C sharp). Fitur dan bentuknya sama kaya NetBeans dan Microsoft Visual Studio, intinya MonoDevelop bisa jadi alternatif IDE untuk pengembangan C#, F#, Java, Boo, Visual Basic.NET, Oxygene, CIL, Python, Vala, C and C++.

Begini cara install di Ubuntu:

sudo add-apt-repository ppa:keks9n/monodevelop-latest
sudo apt-get update
sudo apt-get install monodevelop-latest

Sudah selesai?
