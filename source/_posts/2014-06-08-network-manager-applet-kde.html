Beberapa minggu yang lalu install KDE di notebook kesayangan, tapi masih jarang dipake karena masih belum lengkap, salah-satunya network applet atau network manager. Bukannya langsung install malah bolak-balik KDE - GNOME. Akhirnya googling dan KDE Plasma harus install dulu, coba beberapa cara tapi ternyata gak bisa. Awalnya mau coba install paket network-manager-kde tapi ternyata paket tidak ditemukan, pantang menyerah terus googling lalu nemu paket yang harus diinstall ternyata plasma-widget-networkmanagement, yaudah langsung install saja dengan perintah

$ sudo apt-get install plasma-widget-networkmanagement

Tapi setelah diinstall kok gak muncul di system tray? Oh ternyata harus munculin manual, caranya klik kanan di System tray -> System Tray Settings -> Display -> centang Network Management, muncul deh.

Nanti muncul di system tray-nya seperti ini, ada logo wifi-nya kalau konek ke wifi

Dan ini tampilan menu-nya ketika diklik
