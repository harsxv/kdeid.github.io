Pasti udah sering denger kan istilah Live CD? Itu lho, CD yang isinya installer distro linux tapi kita bisa pake distronya tanpa harus install dulu, jadi CD/DVD langsung pake aja. Nah, sekarang kalau cuma punya netbook tanpa ada optical drive gimana? Ya pake UFD alias USB Flash Disk dong :D

Ada beberapa cara, misalnya pake unetbootin atau pake dumdisk. Pake unetbootin itu gampang, soalnya ini aplikasi portable dan punya GUI, tapi kadang gak berhasil boot ke USB kalau kita lupa kasih label boot ke UFD-nya. Biasanya sih dulu sering pake unetbootin, tapi sekarang nemu cara yang lebih cepet, lebih gampang juga.
Bahan:

  1. Siapin dulu USB Flash Disk, terserah mau ukuran berapa, ya paling nggak 4 GB udah lebih dari cukup.
  2. File ISO installasi linux.
  3. Harus diperhatikan juga speed UFD-nya, ini menentukan speed tulis dan baca, merek juga penting. Disarankan sih transcend atau sandisk, kalau ada kingston enaknya seri baru semacam G-101.
  4. OS Linux, kenapa linux? Soalnya dumpdisk cuma ada di linux, disini pake ubuntu sih.

Langkah:

  1. Backup data di UFD terus format, ke 32 aja, tujuannya cuma buat kosongin space aja.
  2. Buka terminal, terus ketik ini: 
    sudo dd if=/path-to-iso/iso of=/lokasiUFD

    Misal:

    sudo dd if=~/documents/iso/ubuntu-12.04-amd64.iso of=/dev/sdb
  3. Tunggu sampai selesai.
  4. Restart komputer.
  5. Masuk ke BIOS -> Setting boot priority pertama ke UFD
  6. Exit & Save changes di BIOS
  7. Boot ke USB

Selesai? Selamat mencoba live USB-nya dan jangan lupa belajar buat UAS!
