@SENDYYeah om, pentest di ubuntu ada ndak ? Tau repo nya ?

Carane piye? Sini om bisikin!
Caranya, buka https://launchpad.net/ terus cari Kali Linux di kolom search.

Klik repositori yang pertama atau yang paling atas (Kali Linux Tools PPA 1 : wagung.

Nanti di repository pertama, klik tulisan "Technical details about this PPA" ada link ke repository lain, buka juga Kali Linux Tools PPA 2 dan Kali Linux Tools PPA 3.

Nah, sekarang kumpulin informasi ppa nya apa aja dari ketiga repository tadi. Ini list-nya:

  • ppa:wagungs/kali-linux
  • ppa:wagungs/kali-linux1
  • ppa:wagungs/kali-linux2

Sekarang buka terminal, lalu ketik perintah ini:

$ sudo add-apt-repository ppa:wagungs/kali-linux
$ sudo add-apt-repository ppa:wagungs/kali-linux1
$ sudo add-apt-repository ppa:wagungs/kali-linux2
$ sudo apt-get update

Sudah selesai? Sekarang silakan buka synaptic package manager buat install tools-nya. Belum punya?

$ sudo apt-get install synaptic

Atau bisa juga dilihat di Ubuntu Software Center, biasanya masuk ke situ juga kalau gak salah. KALAU GAK SALAH :p

OK, selamat mencoba dan jangan sembarangan pake toolnya.
