Mate (Dibaca 'mate' bukan 'met' karena berasal dari kata matey) Desktop akan tersedia di repository Ubuntu 14.04 Trusty Tahr dan user bisa menginstallnya langsung via Ubuntu Software Center.

Mate Desktop merupakan desktop environment berbasis Gnome 2 dengan tampilan "Gnome Classic" sebelum Ubuntu mengembangkan Unity Desktop. Mate Desktop merupakan alternatif desktop environment untuk pengguna yang kangen dengan tampilan klasik Gnome 2 dan gagal move-on ke Gnome 3 :D

Mate sudah diupload ke repository 'Trusty Proposed' untuk tahap pengetesan. Jika tidak ada kendala, Mate akan tersedia untuk pengguna 'Trusty Testing' atau Trusty Daily build di Ubuntu Software Center segera.
