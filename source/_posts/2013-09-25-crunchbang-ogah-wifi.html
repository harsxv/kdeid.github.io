Baru install CrunchBang, terus dibawa ke kampus niatnya biar bisa wifi-an pake akun mahasiwa, ternyata kandas ditengah ruang serbaguna :| Setelah googling sana-sini dengan bantuan USB tethering dari HP jadul, ternyata masalah utamanya adalah si debian gak install driver wireless LAN dalam paketannya, akhirnya cari dulu ini WLAN pake chipset apa dengan perintah 'lspci' dan menemukan si neneg pake Broadcom.

Terus kalau ngikutinÂ https://wiki.debian.org/wl, cara pertama untuk debian wheezy adalah:

deb http://http.debian.net/debian/ wheezy main contrib non-free

Tapi karena ini CrunchBang, cara pertama itu bisa di-skip :D

Selanjutnya update repo terus download driver Broadcom Wireless

# apt-get update
# apt-get install linux-headers-$(uname -r|sed 's,[^-]*-[^-]*-,,') broadcom-sta-dkms

Terus unload modul yang nantinya bisa menimbulkan prahara dan konflik

# modprobe -r b44 b43 b43legacy ssb brcmsmac

Terus ganti pake modul wl

# modprobe wl

Selesai, langsung bisa ngacir wifi-an :D
