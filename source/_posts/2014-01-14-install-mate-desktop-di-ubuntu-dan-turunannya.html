Kalau di Ubuntu 14.04 Trusty Tahr si Mate Desktop ini udah masuk repository testing, gimana dengan 12.04 atau pengen install karena gagal move on dari Gnome 2 ke Gnome 3 :p

Sekarang, mari install Mate Desktop biar bisa bernostalgila :D

Buat Ubuntu 12.04 LTS Precise Pangolin, buka terminal lalu tambahkan satu aja list repo atau satu perintah saja dari tiga perintah dibawah.

$ sudo add-apt-repository "deb http://packages.mate-desktop.org/repo/ubuntu precise main"
$ sudo add-apt-repository "deb http://repo.mate-desktop.org/ubuntu precise main"
$ sudo add-apt-repository "deb http://mirror1.mate-desktop.org/ubuntu precise main"

Inget ya, satu aja dan gak usah ketiganya.

Buat Ubuntu 13.10 Saucy Salamander, tinggal ganti precise diatas jadi saucy. Jadinya begini, salah satu ya.!

$ sudo add-apt-repository "deb http://repo.mate-desktop.org/ubuntu saucy main"
$ sudo add-apt-repository "deb http://packages.mate-desktop.org/repo/ubuntu saucy main"
$ sudo add-apt-repository "deb http://mirror1.mate-desktop.org/ubuntu saucy main"

Kalau udah, update repository list dan autentikasi mate-archive-keyring, caranya? Ini:

$ sudo apt-get update
$ sudo apt-get --yes --quiet --allow-unauthenticated install mate-archive-keyring
$ sudo apt-get update

Terus install mate desktop, kalau mau install base package atau cuma desktop dan beberapa aplikasi inti, install dengan perintah:

$ sudo apt-get install mate-core

Nah, kalau mau komplit pake telor dan kornet alias install juga beberapa paket pendukung, install dengan cara:

$ sudo apt-get install mate-desktop-environment

Sekarang, gimana caranya buat distro lain yang bukan turunan ubuntu? Gampang, buka aja wiki mate disini

Oke, selamat mencoba dan selamat bernostalgia :)
