Setelah berpetualang dengan CSS Framework, akhirnya gue merangkum beberapa framework yang memang menurut gue "terbaik". Penilaian berdasarkan dokumentasi, tingkat kemudahan dalam implementasi, fitur.

  1. Twitter Bootstrap

    [caption id="attachment_2668" align="alignnone" width="625"] Twitter Bootstrap[/caption]

    Twitter Bootstrap dikembangkan oleh developer twitter, dan boleh dibilang semua tampilan yang ada di halaman Twitter ada di Twitter Bootstrap (baik ya Twitter :p ). Selain layout, ada juga fitur responsive design, serta styling untuk elemen lain, seperti button, input form, slider, progress bar, navigation bar, icon set, dropdown, dan masih banyak lagi.

    Nah, hal diatas yang membuat gue menempatkan Twitter Bootstrap di posisi teratas, karena fiturnya banyak, juga dokumentasi yang tersusun dengan rapi dan mudah dimengerti. Gue yang baru nyobain aja langsung jatuh cinta.

  2. Skeleton

    [caption id="attachment_2666" align="alignnone" width="625"] Skeleton[/caption]

    Skeleton ini filenya kecil dan ringan, dokumentasinya juga bagus dan lengkap. Selain itu juga fitur tambahan standard seperti link, form, button juga ada. Jadi kalau ingin edit elemen, tinggal bongkar CSS-nya dan gak usah buat dari awal. Yang pasti responsive ready :p.

  3. SimpleGrid

    [caption id="attachment_2665" align="alignnone" width="625"] SimpleGrid[/caption]

    Kalau yang ini CSS Framework pertama yang gue coba. Kesannya simple dan gampang dipake, karena dokumentasinya juga udah simple dan mudah dimengerti, dengan catatan semuanya dibaca dengan teliti ;) Dan ini juga udah bisa bikin responsive design.

  4. 960 Grid System

    [caption id="attachment_2669" align="alignnone" width="625"] 960 Grid System[/caption]

    Tenar dengan nama 960.gs karena domain mereka begitu, yang sebenernya dokumentasinya sangat lengkap, fiturnya bagus, dan nilai plus buat 960.gs ini ada sketch sheets berformat PDF. Jadi kalau mau bikin sketsa dulu buat bikin web, cukup print file PDF-nya lalu bikin sketsa disitu.

    Cuma gue merasa ribet pas mau bikin responsive design, harus pake adapt.js! Merasa ribet aja kalau udah nemu yang namanya javascript (padahal twitter bootstrap juga pake jQuery).

    Tapi gue akui, 960.gs ini powerful!

Sekian review, coretan, atau apalah tentang CSS Framework, dan Matur Thank You.

Nah, sekarang menurut kalian, CSS Framework yang mana yang terbaik.?
