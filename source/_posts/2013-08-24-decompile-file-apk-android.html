Decompile file APK android, atau extract source code dari file APK bisa gak sih? Ya bisa lah! Sebenernya kalau cuma mau lihat isi yang sebatas file name tinggal buka aja pake archive manager, tapi kalau buka source code-nya ya gak bisa, kan udah dicompile.

Jadi caranya gini, download dulu apktool disini https://code.google.com/p/android-apktool/downloads/list terus copy ke folder /usr/local/bin dan kasih permission 755 +x (yang pasti root required). Selanjutnya siapin bahan-bahan ini:

File framework-res.apk copy dari folder /system/framework/
File apk yang mau di-decompile, ambil contoh file SystemUI.apk dari folder /system/app/
Terminal

Buka terminal, copy juga file framework-res.apk dan file yang mau di decompile di folder apa aja terserah, terus mount di terminal.

Anggap aja ini udah mount di folder file-file tadi ya

Pertama, install dulu framework compiler yang kita ekstrak dari framework-res.apk

$ apktool if framework-res.apk

Terus disitu ada hasil prosesnya, kalau gak ada pesan error kita lanjut ke decompile file apk. Coba cek folder /home/username/framework/

$ apktool -d SystemUI.apk

Kalau berhasil, ada folder baru di /home/username/NamaFileAPK, misal di /home/username/SystemUI.

Sampai disini dulu, selamat mencoba :D *recompile nanti dibahas lagi*
