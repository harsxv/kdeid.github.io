Habis install xfce dan gak tau file konfigurasi apa yang kehapus, soalnya sempet hapusin konfigurasi openbox :p, hasilnya tombol volume di notebook gak bisa dipake. Pasrah dengan keadaan, setting manual terus via audio mixer, lalu googling-googling dapet caranya. Ternyata cuma nambahin keyboard shortcut aja.

Caranya gimana.?
Buka Menu -> Settings -> Keyboard -> Application Shurtcuts -> Add
Isi command ini buat Volume Up:

amixer set Master 5%+

terus isi keyboard shortcut-nya.

Isi command ini buat Volume Down:

amixer set Master 5%-

terus isi keyboard shortcut-nya.

Close terus cobain deh :D
