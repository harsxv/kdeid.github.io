Gara-gara koleksi ISO, musik dan pilem gak jelas, akhirnya partisi /home penuh juga, cuma nyisain 500 MB dan ini menyiksa dan bikin lemot. Akhirnya migrasi ke Fedora 20 biar ringan :D (apa hubungannya?).

Setelah migrasi selesai, saatnya resize partisi. Terus install gparted dari Software manager, eh ternyata gak bisa resize atau unmount sekalipun karena 'volume is busy' (iyalah, lagi dipake itu harddisknya). Googling-googling ternyata harus pake live CD atau pake gparted live yang bisa didownload disini http://gparted.org/livecd.php tapi downloadnya gede amat, 127 MB kalau gak salah.

Ditengah frustasi tadi, tiba-tiba nengok UFD yang masih nempel di port USB laptop, dan punya ide buat jalanin live session fedora lagi lalu install gparted di live session. Pinter kan? :D

[caption id="attachment_3486" align="alignnone" width="640"] Ini dia tersangkanya![/caption]

  • Jadi pertama-tama baca basmallah dulu, terus restart system terus pilih boot ke USB, kalau pake DVD / CD ya boot ke Optic Drive, terus masuk ke Live Session / Live CD atau apalah namanya.
  • Buka software center, kalau di ubuntu namanya Ubuntu Software Center, di fedora namanya Software, menyesuaikan lah.
  • Install gparted (bisa juga aplikasi partition editor yang lain).
  • Buka gparted (atau sejenisnya).
  • Unmount semua partisi yang mau di-resize.
  • Resize masing-masing partisi, contoh sda7 dari 100 GB di-resize jadi 60 GB, terus sda5 dari 100 GB jadi 140 GB.
  • Klik tombol apply.
  • Tunggu proses selesai terus restart dan boot ke HDD.
  • Cek hasil dan jangan lupa ucap hamdallah :D
