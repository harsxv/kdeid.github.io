Karena keranjingan chat via IRC dan keracunan terminal juga, kenapa gak gabungin keduanya? Baiklah! Hasil googling nemu WeeChat dan IRSSI, sekarang mau bahas WeeChat dulu.Â WeeChat ini bukan messenger yang ada di henpon yang bisa lemparin botol, tapi IRC client buat terminal, khususon terminal linux. Jadi chatting-nya text mode, gak seperti di pidgin yang ada GUI nya, ini pake terminal, biar greget kayak heker!

Install dulu WeeChat via package manager, bisa aptitude, yum, pacman, atau ngibrit ke software manager masing-masing. Karena sekarang si gue pake linux mint dan otomatis pake apt, jadi perintahnya demikian:

$ sudo apt-get install weechat

Sudah selesai, jalankan weechat via terminal dengan perintah:

$ weechat-curses

Kalau tampilan weechat yang seperti ini sudah muncul, langsung aja kirim perintah dengan command ala IRC, misal /connect chat.freenode.net lalu Enter

Setelah konek ke IRC server, tinggal join deh ke kanal alias channel, misal buat nongkrong di kanal #ubuntu-indonesia tinggal ketik perintah /join #ubuntu-indonesia dan jangan lupa Enter
