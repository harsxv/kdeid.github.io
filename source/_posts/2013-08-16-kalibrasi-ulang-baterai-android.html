"Batre ponsel android anda boros? Full charge tapi cuma bisa kepake 5 menit? Tenang saja, saya beri anda solusinya!" pernah denger gak di TV iklan kek gitu.? Nggak kan ya.? yaudah, cuma disini yang ada begituannya. Biasanya batre boros itu karena emang batrenya udah soak, konsumsinya emang boros, atau bisa juga ada kesalahan di statistik penggunaan batre, yang sekarang mau dibahas itu karena battery stat nya ngaco.

Caranya gimana biar gak ngaco? Beli baru? nggak, Terus? Kalibrasi ulang aja baterai android-nya :D

Caranya kalibrasi ulang? Nih!

  • Android udah di root, ini penting. Hari gini android belum di root? Biasa aja ah.
  • Install Root Explorer disini https://play.google.com/store/apps/details?id=com.speedsoftware.rootexplorer (beli sih 4 dolaran) atau ES File Explorer disini https://play.google.com/store/apps/details?id=com.estrongs.android.pop (gratis tapi cukup bagus).
  • Charge battery sampe 100%, biarin dulu sampe batre mateng (gak deng).
  • Udah full charge, buka File manager yang tadi, terserah mau Root Explorer atau ES File Explorer.
  • Mount ke directory /data/system
  • Cari file batterystat.bin
  • Hapus file batterystat.bin
  • Cabut charger.
  • Pake androidnya sampe batre bener-bener habis, sampe device-nya mati sendiri, disini bakal bikin batterystat.bin yang baru.
  • Charge ulang sampe 100% dan jangan sampe keputus, pake biasa deh.

Kalibrasi ulang wajib kalau install atau update rom / firmware baru, pake custom rom, atau batre mulai ngaco.
