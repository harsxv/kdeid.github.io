Udah lama kenal bootstrap, sekitar 2 bulanan lah ya. Jadi karena udah cukup akrab dengan CSS Framework besutan twitter ini, kadang merasa ada yang kurang, apa ya? Misal icon kurang banyak, warna kurang menarik, ini itu ini itu. Nah sekarang bahas aja deh Resource yang udah pernah dipake.

  1. Font Awesome

    [caption id="attachment_2749" align="alignnone" width="625"] Font Awesome[/caption]

    Awalnya gue kira ini font library mirip-mirip google fonts, tapi ternyata isinya kumpulan icon tapi berbentuk font, apa font yang bentuknya icon? Tapi yang pasti, icon disini lebih banyak dari icon bawaan twitter bootstrap, formatnya vector, jauh lebih tajam dari yang cuma PNG, dan retina display ready.

  2. Bootswatch

    [caption id="attachment_2752" align="alignnone" width="625"] Bootswatch[/caption]

    Bootswatch itu semacam theme gallery buat twitter bootstrap, isinya custom CSS dari twitter bootstrap yang menghasilkan theme baru. Baru sedikit sih memang, tapi cukup buat memberi inspirasi tampilan apa yang nanti akan dibuat dari twitter bootstrap.

  3. Bootstrap Image Gallery

    [caption id="attachment_2754" align="alignnone" width="625"] Bootstrap Image Gallery[/caption]

    Ini kalau butuh Image Gallery, resourcenya bisa dipake buat nampilin gambar dengan gallery view. Gak cuma grid view, ada juga fitur slideshow dengan modal view. Pokoknya udah mantap lah Image Gallery nya.

  4. Bootstrap Date Picker

    [caption id="attachment_2755" align="alignnone" width="625"] Bootstrap Date Picker[/caption]

    Karena HTML 5 dan datepicker-nya punya tampilan yang biasa aja, standard, kadang gak jalan di beberapa browser, datepicker cukup lumayan buat mengatasi masalah-masalah beginu. Tinggal baca dokumentasi, pasang, selesai.

Yaudah sih, gitu doang.. Ada yang kurang? Nanti deh ditambah lagi.
