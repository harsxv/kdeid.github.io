Kalau yang sedang berkutat dengan file style.css dan gak mau ganggu bootstrap.css yang memang kalau lagi develop gak boleh diganggu, terus punya masalah karena styling CSS punya kita bentrok sama punya bootstrap dan hasilnya styling gak dipake browser, tapi tetep pake styling bawaan bootstrap, caranya gampang sih.

Kalau inget "!important" yang biasa disispkan di value dari tiap property CSS, pake aja itu, ya emang itu sih yang sekarang dibahas. *Gak ada bahasan lain, soalnya lagi males bongkar-bongkar bootstrap di UFD.

Jadi gimana sih maksudnya?

Kalau kamu punya CSS macam gini:

.navbar {
font-family: ubuntu;
font-color: white;
}

tapi styling gak mau load, tambahi !important sebelum titik koma. Contoh:

.navbar {
font-family: ubuntu !important;
font-color: white !important;
}

Insya allah dengan seijin tuhan dan sekemampuan browser, custom styling bisa loaded sempurna.
