Cerita pengalaman di Arch lagi, kenapa? Karena ternyata di arch gak semuanya serba otomatis macam di ubuntu. Sudah tiga hari sejak install Gnome Music, kok kosong terus ya... Google-kena-google, ternyata file di home nggak ke-track secara otomatis, jadinya file yang ada di folder Music gak masuk ke Gnome Music.

Caranya gimana? Gini:

$ tracker-control -e
$ tracker-control --start

Untuk melihat statusnya:

$tracker-control --status

Terus tunggu hingga semua statusnya ada tanda centangnya, terutama di File System.

Selanjutnya restart Gnome Music dan tunggu sampai proses import-nya selesai :D
