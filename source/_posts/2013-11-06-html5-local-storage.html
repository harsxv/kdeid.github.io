HTML5 Local Storage itu apa bero? Itu tuh, jadi kisa bisa nyimpen data di storage lokal atau singkatnya di dalam browser. Bedanya sama cookies? Hampir sama, sama-sama nyimpen data, tapi local storage lebih unggul dari besarnya kapasitas yang bisa sampe 5 - 10 MB, kalau cookies cuma 4kB.

Browser yang mendukung:

Udah banyak browser yang mendukung, kalau dilihat di sini sih, banyaaaak.... Kalau mau main aman, jangan lupa pake logika if :D

if (window["localStorage"]) {
    // your localStorage code here... 
}

Contoh kode:

Gak jauh-jauh sama kode deh disini, coba cek beginian

localstorage.setItem("storage1","value 1"); 
var value=localstorage.getItem("storage1");
alert(value);

Jadi Local Storage ini bisa nyimpen data dengan method setItem, dan ngambil data dengan getItem, selain itu juga key atau nama dari item harus unik, gak boleh bentrok atau bertumpuk dengan nama lain.

Ini contoh yang salah:

local.storage.setItem("storage1","value 1");
local.storage.setItem("storage1","value 2");
var value = localstorage.getItem("key1");
alert(value);

Gak bisa disebut salah juga sih, tapi kalau kode itu dieksekusi, yang tampil untuk storage1 adalah value 2, karena value 1 udah gak dianggap lagi setelah ditimpah value 2.

Local Storage & Array

Selain pake setItem dan getItem, local storage juga bisa pake array, contohnya:

localStorage["storage1"]="value 1";
var value=localStorage["storage1"];

Nah, sekian dulu tentang Local Storage, selamat mencoba :D
