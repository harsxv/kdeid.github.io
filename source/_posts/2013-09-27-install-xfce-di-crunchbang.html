Buat yang merasa ribet dengan obmenu di openbox, terus harus nambahin menu sendiri pake menu editor, mending pindah ke Desktop Environment yang lain seperti LXDE yang juga jalan di OpenBox atau xfce. Nah, disini aku pilih xfce biar lebih cantik dari openbox atau lxde. Sebenernya OpenBox dengan tint2 nya masih bisa lebih cantik, tapi males ngopreknya :D

Gimana cara install xfce di CrunchBang? Gini:

$ sudo apt-get install xfce4

Terus kalau perlu install beberapa utility dari xfce, install juga xfce-goodies

$ sudo apt-get install xfce4-goodies

Ini perbandingan memory usage antara openbox dan xfce:

[caption id="attachment_3222" align="alignnone" width="656"] Resource Usage OpenBox + tint2[/caption]

[caption id="attachment_3223" align="alignnone" width="648"] Resource Usage xfce4[/caption]

Gak beda jauh penggunaannya, tapi kalau setiap Byte memory kamu begitu berharga, lebih baik pilih OpenBox + tint2 bawaan CrunchBang atau malah tanpa DE sama sekali :p

Selesai install, pindahin default session dari openbox ke xfce.

$ sudo update-alternatives --config x-session-manager

Terus isi password, pilih nomor yang ada xfce session-nya, logout lalu login lagi.
