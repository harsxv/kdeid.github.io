Ada yang gak tau aplikasi messenger LINE? Kebangetan! Nah, kemaren si gueh coba install naver line di linux ubuntu 12.04 pake wine, tapi ternyata gak bisa, muncul error yang katanya LINE udah running. WHAT? Yaudah, googling-googling ternyata harus install Visual C Runtime, terus install deh pake winetricks yang perintahnya kayak begini:

$ winetricks vcrun2008

Terus coba install lagi, eh ternyata gak bisa. Akhirnya nemu video tutorial di youtube katanya harus copy hasil install di windows. DUH.

Akhirnya download dulu deh hasil install yang dia punya dari sini: http://db.tt/G5doCAO1

Terus ikutin deh langkahnya disini:

  1. Pertama, harus ada wine, cara installnya gimana? Nih 
    $ sudo apt-get install wine
  2. Terus install visual C runtime pake winetricks 
    $ winetricks vcrun2008
  3. Download file di link diatas tadi atau link ini: http://db.tt/K1yslM4w
  4. Extract hasil download.
  5. Copy folder LINE ke /home/username/.wine/drive_c/Program Files
  6. Terus buat file baru dengan nama line di /usr/bin dengan nama line 
    $ sudo gedit /usr/bin/line
  7. Copy string ini ke file: 
    #!/bin/bash
wine /home/sendz/.wine/drive_c/Program Files/LINE/Line.exe 2>&1
/dev/null
  8. Save, jalankan line via terminal, ketik line terus enter.
  9. Beres deh.
