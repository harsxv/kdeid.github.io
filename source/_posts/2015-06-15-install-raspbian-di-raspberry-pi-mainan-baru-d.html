Malam ini cukup membahagiakan, setelah berkunjung ke kosan Fahmi dan dia curhat galau Tugas Akhir yang tak kunjung selesai, akhirnya saya berniat membantunya, dan yang membuat saya bahagia adalah saya mendapat Raspberry Pi A yang sudah lama saya idamkan tapi belum ada rezeki untuk memboyong satu unit ke kosan.

Tugasnya 'cuma', (gak mau meremehkan sih) bikin script PHP yang bisa print langsung ke printer via raspi, kenapa? Karena web servernya nanti ada di raspi, begitu ceritanya.

Nah, saya dapet raspi A kosongan, belum install apa-apa bahkan SD Card yang Fahmi kasih belum ada OSnya. Baiklah.

Sekarang yang akan saya ulas adalah cara install Sistem Operasi di Raspberry Pi, saya menggunakan Raspbian Wheezy.

  1. Download file Raspbian dari sini https://www.raspberrypi.org/downloads/
  2. Extract file yang telah didownload tadi, cari file dengan ekstensi .img lalu rename menjadi raspbian.img untuk memudahkan proses selanjutnya.
  3. Masukkan SD Card ke USB Card Reader / Card Reader yang tersedia di laptop (notebook / netbook) lalu cek lokasi SD Card, misal di /dev/sdb
  4. Buka aplikasi terminal di linux lalu buka folder di mana file raspbian.img tadi disimpan dengan perintah cd.
  5. Ketikan perintah sudo dd if=raspbian.img of=/dev/sdb untuk mengekstrak raspbian ke SD Card.
  6. Setelah proses dump disk selesai, pindahkan SD Card ke slot yang tersedia di Raspberry Pi.
  7. Nyalakan Raspberry Pi.
  8. Langkah selanjutnya adalah konfigurasi Raspbian, tidak perlu melakukan partisi karena sudah mengikuti pengaturan dari Raspbian, yang saya lakukan hanya mengganti password. Dialog ini bisa dimunculkan kembali dengan perintah sudo raspi-config
  9. Klik tombol Finish di dialog untuk menyelesaikan konfigurasi, lalu system akan merestart sendiri.
