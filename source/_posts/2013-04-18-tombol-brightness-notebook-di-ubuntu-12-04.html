Jadi ceritanya punya notebook baru nih.. ihirrrrr.... si tombol brightness di keyboard itu gak mau fungsi.. Itu lho kombinasi tombol Fn + Panah. Jadi gimana caranya biar bisa fungsi.? Gini

$ sudo gedit /etc/default/grub

*itu tanda dollar jangan ikutan diketik!

Terus cari baris ini:

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"

Tambah acpi_backlight=vendor setelah quite splash jadinya nanti seperti ini

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor"

Terus filenya nantinya seperti ini

# If you change this file, run ‘update-grub’ afterwards to update
# /boot/grub/grub.cfg.
# For full documentation of the options in this file, see:
# info -f grub -n ‘Simple configuration’

GRUB_DEFAULT=0
GRUB_HIDDEN_TIMEOUT=0
GRUB_HIDDEN_TIMEOUT_QUIET=true
GRUB_TIMEOUT=10
GRUB_DISTRIBUTOR=`lsb_release -i -s 2> /dev/null || echo Debian`
GRUB_CMDLINE_LINUX_DEFAULT=”quiet splash acpi_backlight=vendor”
GRUB_CMDLINE_LINUX=”"

$ sudo update-grub

Restart, dan cobain tombolnya. Kalau berhasil pasti berubah brightness layarnya. :D
