Sudah dapat GitHub Education Pack? Kalau sudah, kita bisa ambil offering dari GitHub dan sponsor lain. Caranya? Buka https://education.github.com/pack/ (Kalau belum, bisa buka post saya di sini GitHub Education Pack untuk meminta diskon atau hak untuk offering dari GitHub).

Karena hari ini saya baru saja redeem voucher DigitalOcean senilai 100 USD, yep, 100 Dollar Amerika, senilai Rp. 1.330.000,- yang bisa digunakan untuk membayar VPS yang disediakan oleh DigitalOcean. DigitalOcean sendiri adalah penyedia Virtual Private Server (VPS) yang juga menjadi sponsor GitHub Education.

Lanjut saja, begini caranya:

Request Coupon Code di GitHub Education

  1. Buka GitHub Education Pack di https://education.github.com/pack/ lalu klik "Get your pack".

    [caption id="attachment_3956" align="alignnone" width="700"] Get your pack![/caption]

  2. Scroll ke bawah hingga menemukan DigitalOcean, lalu klik link "request coupon code" atau semacamnya, saya lupa nama link-nya.

    [caption id="attachment_3957" align="alignnone" width="700"] Request Kode Kupon[/caption]

  3. Tunggu hingga proses verifikasi selesai dan coupon code didapatkan.

Redeem Coupon Code di DigitalOcean

  1. Akses https://www.digitalocean.com/ lalu pilih Log In jika sudah memiliki akun, atau Sign Up untuk mendaftarkan akun baru.

    [caption id="attachment_3958" align="alignnone" width="700"] DigitalOcean[/caption]

  2. Klik icon gear lalu masuk ke menu settings lalu pilih Billing (https://cloud.digitalocean.com/settings/billing)
  3. Kita perlu melakukan aktivasi akun DigitalOcean dengan cara menambahkan kartu kredit atau melakukan deposit minimal 5 USD via PayPal.

    [caption id="attachment_3959" align="alignnone" width="700"] Tambahkan Kartu Kredit atau Deposit[/caption]

  4. Setelah deposit berhasil dan akun aktif, masukkan Coupon Code yang didapat dari GitHub Education tadi dan tunggu hingga proses verifikasi selesai, maka credit senilai USD 100 akan ditambahkan secara otomatis dan bisa kita gunakan untuk membuat droplet VPS.

    [caption id="attachment_3960" align="alignnone" width="700"] Redeem Code[/caption]

    Kotak biru adalah bukti kalau saya sudah redeem kupon senilai USD 100, dan kotak merah adalah tempat untuk memasukkan kode kupon.

Ini adalah summary account saya saat ini, masih tersimpan USD 105 yang bisa saya gunakan untuk menyewa layanan dari DigitalOcean.
