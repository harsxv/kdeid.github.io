"Throw away the graphical user interface!" kata sebuah judul post di suatu website. Cocok! Entah kenapa nyari aplikasi buat terminal itu bagai sebuah ambisi yang tidak bisa dihentikan, ada rasa bangga bisa browsing, muter musik, chatting, atau bahkan twitteran dengan layar hitam text putih ini, rasanya pengen bilang "GUE PUAAAAASSSS...!!!". Am I freak? No. Am I geek? No. Am I nerd? No. So? I just linux user yang sedang keranjingan terminal.

Setelah nulis mutt, moc, weechat dan beberapa aplikasi lain yang running di terminal, sekarang mari coba earthquake buat twitteran di terminal. Bukan twitteran di terminal bis lho ya.

Nah sekarang mari kita install si cantik earthquake ini. Kalau merujuk ke project page-nya disini https://github.com/jugyo/earthquake, earthquake ini pake ruby, jadi sebelum install earthquake, install ruby dan rubygems dulu. Kalau sudah tinggal install earthquake via gem.

$ gem install earthquake

Kalau sudah, buka earthquake dengan perintah:

$ earthquake

Untuk pertama kali, earthquake butuh access PIN ke akun twitter, cara mendapatkan PIN cukup klik link yang muncul, authorize access earthquake dan salin PIN ke konfigurasi earthquake, setelah selesai earthquake akan menampilkan timeline.

Sekarang, gimana cara pake earthquake?

  1. Ketik perintah :help kalau bingung dan butuh pertolongan.
  2. Untuk mengirim tweet, ketik text disebelah tanda petir (⚡) lalu enter, terus konfirmasi, y / n lalu tekan enter lagi.
  3. Untuk membalas tweet atau reply tweet, lihat tanda yang ada di dalam bracket, isinya karakter dollar dan dua huruf/angka, itu sebagai tweet ID, jadi ketika reply / retweet, cantumkan ID tersebut. Misal untuk reply: 
    ⚡ $ab Test Reply
  4. Untuk retweet: 
    ⚡ :retweet $ab
  5. Untuk menghapus tweet: 
    ⚡ :delete $ab
  6. Load timeline: 
    ⚡ :recent
  7. Load mention 
    ⚡ :mentions
  8. Untuk selengkapnya bisa lihat menu help tadi
