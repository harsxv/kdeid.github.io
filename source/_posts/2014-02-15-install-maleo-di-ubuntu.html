Maleo itu apa? Burung yang hampir punah asal Sulawesi. Tapi yang ini beda, maleo ini adalah HTML5 application runner yang dikembangkan dengan bahasa Vala, jadi si aplikasi HTML5 gak perlu load lagi di browser.

Gimana cara installnya? Pertama-tama, karena software ini membutuhkan dependensi dari gnome, install dulu paket gnome-common.

sudo apt-get install gnome-common

Lalu download dulu maleo disini https://github.com/mdamt/maleo/archive/master.zip, selanjutnya ekstrak file tersebut di working directory lalu mount via terminal.

Setelah folder dibuka via terminal, install maleo dengan menjalankan file autogen.sh

./autogen.sh

Tunggu proses install selesai. Tutorial penggunaan bisa dilihat disini https://github.com/mdamt/maleo
